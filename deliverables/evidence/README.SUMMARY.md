# Evidence Pack — Executive Summary

> Public summary of `README.md` for the tricura-challenger evaluation. Full
> content held privately; available to the evaluator on request.

## Document type and purpose

This directory holds the audit's evidence pack — redacted supporting
artefacts referenced from individual findings in the bug catalog. Each
artefact captures a specific observable that a catalog row cites as
reproduction evidence. The README provides the directory's contribution
conventions: how new evidence is added, how artefacts are named for
cross-reference traceability, and the redaction standard that an artefact
must meet before it may be placed in the public repository.

## Availability

The complete unredacted README is held privately under the candidate's
`.private/` workspace (not committed to the public repository) and is
available to the evaluator for review on request. Individual evidence
artefacts in this directory are present only in their redacted form; the
unredacted originals are similarly held privately.
